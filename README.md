## Installation
1. Clone this repository:
```
git clone https://github.com/AtheerAldosari/Devteams-Web-Application.git
```
2. Install packages
```
cd Devteams-Web-Application
npm install
```
3. Run the backend
```
cd backend
npm run dev
```
4. Run the frontend
```
cd frontend
npm start
```
